Electric Sheep

What androids dream of. Electric Sheep is a web application that plays AI-generated music, accompanied with a unique visualization for every song. New music is continually generated so you'll never run out of music to listen to!

This project was built in 24 hours for HackED 2021. The music generation model used is Google Magenta's Music Transformer (using the released weights). Credit for porting the Colab notebook to a Python script goes to Elvenson on GitHub. The design of this website is based on Google Magenta's Listen to Transformer project. If you'd like to check out the code for Electric Sheep check out the GitHub repo.
